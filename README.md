# Proyek Script AI Sederhana

Proyek ini adalah script AI berbasis Python yang dapat menghasilkan teks otomatis menggunakan model GPT-2 dari Hugging Face.

## Cara Menjalankan di Komputer Lokal

1. **Clone Repositori Ini**
   ```bash
   git clone https://github.com
   cd NAMA-REPOSITORI
   ```

2. **Instal Library yang Dibutuhkan**
   ```bash
   pip install -r requirements.txt
   ```

3. **Jalankan Program AI**
   ```bash
   python app.py
   ```
